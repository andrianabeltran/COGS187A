Visibility of system status

The system should always keep users informed about what is going on, through appropriate feedback within reasonable time.

Home Page

There is generally a consistent layout once you enter the site after the home page, which seems a little more like a landing page.

For the most part, the top navigation bar is consistent, but the subnavigation bar below is either not present or consistent. This is confusing for the user.

Also the "Home" button is named differently - sometimes as "Home" and sometimes as "University Home." It seems like the site is divided and what seems like the landing page is the "University Home," while the "Study with us" page which does not have a subnavigation bar is the "Home." This makes it confusing for the user if they are looking for navigation status of the site.

When the subnavigation bar is present, it is not always consistent either. For example, there is no subtopic for "Current Students". Instead the subtopic is labeled "Learn at Leicester."
